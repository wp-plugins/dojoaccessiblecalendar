Dojo Accessible Calendar is a WAI-ARIA Enabled Calendar Plugin for Wordpress Version 3.3.2

== Description ==

Dojo Accessible Calendar is a calendar of your blog’s posts and uses the dojo WAI-ARIA enabled calendar.

== Installation ==

Upload the Dojo Accessible Calendar plugin to your blog, Activate it, then cutomize it's appearance.

1, 2, 3: You're done!
